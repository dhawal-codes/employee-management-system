# Employee Management System

## Overview
The Employee Management System is a Python application designed to efficiently handle employee records. It offers both a Command Line Interface (CLI) and a Graphical User Interface (GUI) for user interaction. Employee information is saved in a JSON file, ensuring that the data remains intact between sessions.

This project showcases how to implement CRUD operations (Create, Read, Update, Delete), along with features for searching and analyzing employee statistics.

## Purpose of the Project
The goal of this project is to practice and showcase various Python programming concepts, including data management, file handling, command-line interfaces, and GUI development.

## Technologies Used
- Python 3
- Tkinter (for GUI development)
- JSON (for data storage)

## Features
- Add Employee
- View Employees
- Search for Employees by ID or Name
- Update Employee Information
- Delete Employee Records
- Display Employee Statistics
- Data stored in JSON format
- CLI-based menu system
- GUI-based desktop application
